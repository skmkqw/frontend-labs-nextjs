# Frontend Labs

## Funkcje

- Rejestracja, logowanie i weryfikacja konta z informacjami o sesji w pasku nawigacyjnym.
- Panel profilu z edycją danych oraz podglądem statusu użytkownika.
- Lista artykułów pobierana z Firestore na podstawie identyfikatora uzytkownika.
- Gra Scrabble (lokalny tryb): z planszą 15×15.
- Drag & drop na planszy wraz z aktualnym podglądem ruchu i premiami słownymi/literowymi.
- Bieżące komunikaty dla gracza (info/error), podświetlanie ostatniego ruchu i wskazanie aktywnego gracza.

## Uruchomienie

```bash
npm install
npm run dev
# aplikacja startuje na http://localhost:3000
```
